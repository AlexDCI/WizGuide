Следующее задание
Интеграция авторизации пользователя:

Добавьте простую систему пользователей с регистрацией и авторизацией (можно использовать django.contrib.auth).
Реализуйте функционал, чтобы каждый пользователь видел историю своих переводов.
Обновление моделей:

Добавьте связь между пользователем и переводами (например, поле user = models.ForeignKey(User, on_delete=models.CASCADE)).
Интерфейс:

Разработайте простую веб-страницу на Django с формой для ввода текста и выбора языка, которая будет обращаться к вашему API.
Тестирование:

Протестируйте сохранение и отображение данных в базе, включая авторизацию и ограничение доступа к переводам только для их владельца.
